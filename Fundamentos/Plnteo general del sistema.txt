SISTEMA EXPERTO PARA ESTIMACIÓN DE TIEMPOS DE ENTREGA
Distribuidora de Alimentos – Ciudad de Firmat, Santa Fe, Argentina

1. DESCRIPCIÓN GENERAL DEL PROBLEMA
   La empresa distribuidora de alimentos necesita estimar el tiempo de entrega de los pedidos realizados por sus clientes dentro de la ciudad de Firmat.
   Actualmente la estimación se realiza de forma manual e intuitiva, lo que genera:

* Demoras imprevistas
* Mala planificación de rutas
* Falta de criterios claros para priorizar pedidos

Se propone desarrollar un SISTEMA EXPERTO en Prolog e implementando el control de versiones Git (bash) que, mediante preguntas al usuario, permita determinar un tiempo estimado de entrega basándose en variables preconcebidas. 
2. OBJETIVO DEL SISTEMA
   Desarrollar un sistema capaz de:

* Recibir datos de un pedido
* Evaluar variables logísticas relevantes
* Aplicar reglas de inferencia
* Devolver un tiempo estimado de entrega en horas y/o dias

3. VARIABLES PRINCIPALES
   El modelo se basará inicialmente en dos variables centrales:

3.1 Peso del pedido

* Pedidos livianos
* Pedidos medianos
* Pedidos pesados

A mayor peso:

* Mayor complejidad logística
* Posible necesidad de vehículo específico
* Mayor prioridad si se abona extra


3.2 Costo total del pedido

* Pedidos de bajo valor
* Pedidos de valor medio
* Pedidos de alto valor

A mayor costo:

* Posibilidad de entrega prioritaria
* Descuento en tiempo de espera
* Tratamiento preferencial



4. VARIABLES SECUNDARIAS

* Distancia al centro de distribución
* Zona de la ciudad
* Disponibilidad de reparto




5. FUNCIONAMIENTO ESPERADO

El sistema deberá:

1. Realizar preguntas al usuario sobre el pedido

2. Clasificar el pedido según las variables

3. Aplicar reglas de inferencia

4. Mostrar:

   * Tiempo estimado
   * Nivel de prioridad




6. TECNOLOGÍA A UTILIZAR

* Lenguaje: Prolog
* Paradigma: Programación lógica
* Modelo: Sistema experto basado en reglas




7. RESULTADO ESPERADO
    Un prototipo funcional de sistema experto que permita apoyar la toma de decisiones logísticas en la distribuidora.
